Class Detail

FObservable()
Base class for observable objects. Objects inheriting from FObservable may emit signals which may be connected to by other objects. A signal may be any event that the object wants to expose to others. A signal may be emitted via the FObservable#emit function. Use FObservable#connect to connect a callback to a signal.
